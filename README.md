Portfolio of my hands-on network engineering projects using Packet Tracer, showcasing enterprise network design, implementation, troubleshooting, configuration and documentation.


**Project:**
Dual-Site Enterprise Network

**Objectives**
- Hierarchical Enterprise Design
- Secure Site-to-Site Connectivity
- Dynamic Routing
- Public and Private Services
- Redundancy and High Availability
- Layered Security Architecture


**Implemented Features**
- VLAN Segmentation
- VLSM / IP Addressing
- Redundancy via Etherchannels
- GRE over IPsec VPN connectivity
- OSPF and Static routing
- NAT and PAT services
- HTTPS, DNS, DHCP, FTP services
- ASA firewall protection
- ACL-based traffic control
- Layer 2 security (Port Security, DHCP Snooping, DAI)
- Management VLAN isolation
- Secure SSH administration
